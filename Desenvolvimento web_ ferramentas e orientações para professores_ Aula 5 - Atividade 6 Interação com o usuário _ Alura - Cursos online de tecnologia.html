Transcrição

No vídeo anterior, criamos um alerta utilizando o seguinte código para exibir uma mensagem:

alert("Olá mundo");

Criando a função prompt()

No entanto, após reconsiderar, decidimos remover o alerta. Então vamos excluir esta linha de código e substituí-la por outra função chamada prompt(). O prompt é uma função nativa do JavaScript que abre um campo para que o usuário possa inserir um texto.

Entre parênteses, vamos incluir uma pergunta entre aspas duplas: "Qual o seu nome?". É uma boa prática adicionar um ponto e vírgula ao final da linha de código, embora seja opcional na última versão do JavaScript.

prompt("Qual o seu nome?");

Vamos salvar as alterações e visualizar o resultado no navegador. Ao atualizar a página, uma janela com a pergunta é exibida e há um campo de texto para respondermos. Vamos inserir uma resposta como, por exemplo, "Silvio". No entanto, percebemos que nada acontece, pois a resposta não é armazenada em nenhum lugar, então ainda não alcançamos o resultado desejado.

Criando uma variável

Então, vamos retornar ao nosso código JavaScript no Visual Studio Code, no arquivo main.js. Antes do prompt(), na mesma linha, vamos criar uma variável. Mas o que é uma variável? Você pode imaginá-la como uma caixa no computador onde podemos armazenar uma informação e acessá-la por meio do código.

Vamos criar essa variável utilizando uma palavra reservada do JavaScript chamada let. Perceba que o Visual Studio Code formata essa palavra com uma cor diferente, indicando que é uma palavra reservada. let significa "permitir", ou seja, estamos permitindo a criação de uma variável. Agora, essa variável precisa de um nome. Vamos chamá-la de nome-usuário.

let nome-usuário prompt("Qual o seu nome?");

Observe que alguns sublinhados vermelhos surgiram, isso porque o Visual Studio Code verifica erros de sintaxe automaticamente. Portanto, os sublinhados indicam um erro de sintaxe. Um destes sublinhados está exatamente no hífen que usamos. Neste caso, não podemos usar hífens em nomes de variáveis. Em vez disso, podemos usar underlines ou letras maiúsculas para separar palavras em nomes de variáveis. Então, vamos corrigir para nomeUsuário, mantendo o acento. Na última versão do JavaScript, é permitido usar acentos em nomes de variáveis.

let nomeUsuário prompt("Qual o seu nome?");

Para atribuirmos um valor a essa variável, usaremos o sinal de igual. Esse valor será obtido a partir do prompt(), ou seja, da janela do navegador onde a pessoa usuária irá digitar algo. 

let nomeUsuário = prompt("Qual o seu nome?");

Dessa forma, já temos uma variável criada que receberá o valor digitado pela pessoa.

Selecionando o objeto com DOM

No entanto, precisamos alterar o valor do elemento no HTML, substituindo o "Fulano" pelo valor digitado no prompt(). Temos isso marcado em nosso HTML, por volta da linha 13. É o elemento <span> com o atributo id chamado nome-usuário. 

<p>Que bom te ver por aqui, <span id="nome-usuário">Fulano</span></p>

Para isso, precisamos selecionar esse elemento em nosso JavaScript, acessando o DOM. Então, precisamos acessar o que o navegador interpretou e obter esse elemento.

Mas antes, vamos criar outra variável, pois esse elemento será armazenado nela. Novamente, utilizaremos a palavra let, e o nome dessa variável será elemento.

let nomeUsuário = prompt("Qual o seu nome?");
let elemento

Para acessar o DOM, utilizamos a palavra-chave document, que representa o objeto que, por sua vez, representa todo o nosso arquivo HTML. Em seguida, acessamos um método desse objeto usando um ponto. Agora, digitaremos querySelector, que é um seletor de consulta, e abrimos parênteses para passar a identificação do elemento que queremos selecionar.

let nomeUsuário = prompt("Qual o seu nome?");
let elemento = document.querySelector()

No index.html, pegamos o nome do id, que é nome-usuário. Copiamos e colamos dentro dos parênteses, envolvendo com aspas duplas, já que é um texto. 

let nomeUsuário = prompt("Qual o seu nome?");
let elemento = document.querySelector("nome-usuário")

É importante indicar que isso é um atributo de id, um identificador que é único. Por isso, colocamos um caractere de hashtag (#) antes do valor, da mesma forma que fazemos no CSS para selecionar um id.

let nomeUsuário = prompt("Qual o seu nome?");
let elemento = document.querySelector("#nome-usuário");

Agora, salvamos o arquivo teclando "Ctrl + S". Com o código salvo, vamos destrinchar o significado de toda essa linha, que é fundamental:

letelemento=document.querySelector("#nome-usuário")
palavra reservadanomeatribui valor à variávelacesso ao DOMacessa o métodométodo que busca um elementopassagem de parâmetros

Primeiro, criamos a nossa variável usando a palavra reservada let, seguida pelo nome elemento. Em seguida, atribuímos um valor à variável usando o sinal de igual =. Então, acessamos o DOM utilizando a palavra-chave document.

Em seguida, usamos um ponto ., que nos permite acessar um método deste objeto. O método é querySelector, responsável por buscar um elemento dentro do DOM. Por fim, passamos parâmetro entre parênteses. O parâmetro é o valor que o querySelector buscará em nosso código. Neste caso, é o identificador #nome-usuário, onde # significa que estamos selecionando este elemento pelo atributo de id.

Selecionando a variável

Voltando ao Visual Studio Code, vamos selecionar a nossa variável elemento. Então, passamos elemento, que é o nome da nossa variável. Em seguida, vamos acessar uma propriedade chamada textContent, que significa conteúdo de texto. Esse conteúdo de texto é, atualmente, o texto escrito "Fulano". Então, queremos remover isso e colocar o valor da nossa variável. Logo, passamos um sinal de igual = e o nome da variável nomeUsuário.

let nomeUsuário = prompt("Qual o seu nome?");
let elemento = document.querySelector("#nome-usuário");
elemento.textContent = nomeUsuário;

Vamos salvar e ver o resultado. Ao abrir o navegador e atualizar a página, respondemos o prompt com o nome "Silvio". Porém, a mensagem continua sendo "Que bom te ver por aqui, Fulano", e não "Que bom te ver por aqui, Silvio", como esperávamos.

Vamos abrir o arquivo index.html para entender o motivo. Note que o JavaScript está sendo carregado antes da existência do elemento nome-usuário no HTML. Para corrigir isso, movemos a tag <script> que carrega o arquivo main.js para o final do body do HTML (antes do fechamento da tag </body>), garantindo que o JavaScript só seja executado após o carregamento completo do DOM.

<script scr="main.js"></script>
</body>
</html>

Após salvar essa correção, vamos voltar ao navegador, atualizar a página e responder com um nome, como Silvio, por exemplo. Ao fazer isso, vemos que a interação com a página funciona conforme esperado. Agora, a mensagem "Que bom te ver por aqui, Silvio" está com o nome que digitamos, e não mais com "Fulano".
